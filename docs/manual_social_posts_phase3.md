# Phase 3 — Social Media Posts Manual

## Цель
Усилить внешние сигналы для Phase 3 страниц через социальные сети.

## Платформы
- Facebook (основная)
- Instagram
- LinkedIn
- Telegram

## Шаблоны постов

### 1. Новая страница — Маклер в Таллине
**Текст:**
Зачем нужен маклер при продаже квартиры в Таллине? Рассказываем, как стратегический подход отличается от «просто выложить объявление».
→ https://cityee.ee/ru/makler-v-tallinne/

**Хэштеги:** #маклерталлинн #маклер #таллинн #продажаквартиры #cityee

### 2. Новая страница — Продать квартиру в Таллине
**Текст:**
Как продать квартиру в Таллине за 30–45 дней? Ценовой коридор, конкуренты, сильная подача — разбираем стратегию.
→ https://cityee.ee/ru/prodat-kvartiru-v-tallinne/

**Хэштеги:** #продатьквартиру #таллинн #недвижимость #cityee

### 3. Новый кейс
**Текст:**
Продали 3-комнатную квартиру в Kesklinn за 16 дней по запрашиваемой цене. Рассказываем, что сработало.
→ https://cityee.ee/ru/cases/prodali-3-komnatnuyu-kvartiru-v-kesklinn-za-16-dney/

### 4. Районная страница
**Текст:**
Продаёте квартиру в Lasnamäe? Рассказываем о специфике района: конкуренция, типы покупателей и ценовой коридор.
→ https://cityee.ee/ru/tallinn/lasnamae/

### 5. Оценка квартиры
**Текст:**
Сколько реально стоит ваша квартира в Таллине? Объясняем, как определить ценовой коридор перед продажей.
→ https://cityee.ee/ru/ocenka-kvartiry-v-tallinne/

## График публикаций
| День | Платформа | Тип контента |
|------|-----------|-------------|
| Пн | Facebook + Instagram | Кейс или совет |
| Ср | LinkedIn | Экспертная статья / кейс |
| Пт | Facebook + Telegram | Ссылка на новую страницу |

## Правила
- Каждый пост содержит ссылку на соответствующую Phase 3 страницу
- Использовать конкретные цифры: 10+ лет, 300+ сделок, 2%, 30–45 дней
- Не использовать непроверенные данные
- Фото: реальные объекты или профессиональные стоковые
