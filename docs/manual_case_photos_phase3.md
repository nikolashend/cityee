# Phase 3 — Case Photos Manual

## Цель
Подготовить визуальные материалы для 6 кейсов Phase 3.

## Общие требования к фото
- Минимум 5–8 фото на кейс
- Горизонтальная ориентация (16:9 или 4:3)
- Разрешение: минимум 1200×800px
- Формат: JPEG, оптимизированный (quality 80–85%)
- Размер файла: до 200KB после оптимизации
- Имя файла: `case-{slug}-{nn}.jpg` (например `case-kesklinn-3k-01.jpg`)

## Список кейсов и что фотографировать

### 1. 3-комнатная квартира в Kesklinn (78 м², 16 дней)
- Общий вид гостиной
- Кухня с обеденной зоной
- Спальня (главная)
- Вид из окна (локация!)
- Фасад дома / подъезд

### 2. 2-комнатная квартира в Kadriorg (54 м², 24 дня)
- Гостиная после хоумстейджинга
- Спальня
- Вид в сторону парка Kadriorg
- Фасад дома
- Детали интерьера (lifestyle-подача)

### 3. 1-комнатная квартира в Lasnamäe (32.7 м²)
- Общий вид комнаты
- Кухня
- Ванная
- Вид с балкона / из окна
- Фасад дома

### 4. 3-комнатная квартира в Lasnamäe (66.6 м²)
- Гостиная
- Кухня
- Детская / вторая спальня
- Коридор / прихожая
- Фасад дома

### 5. 2-комнатная квартира в Kristiine (49 м², 18 дней)
- Гостиная
- Спальня
- Кухня
- Вид из окна
- Общий вид дома

### 6. 4-комнатная квартира в Tiskre (92.3 м²)
- Гостиная (просторная)
- Кухня-столовая
- Главная спальня
- Детская
- Вид на окружение / зелёная зона
- Фасад / территория

## Оптимизация
- `alt` теги: описательные, с районом и типом квартиры
- Lazy loading: `loading="lazy"` на все кроме первого фото
- WebP версии при возможности
